Zen Pattern Library

Theme: Yoga

The CSS Zen Garden Group Project will use the HTML provided by the site and will remain unchanged. Group Three (Heather Holt, Wesley VerHoef, and Andrew Larsen) will add CSS to create a unique design and have chosen yoga as theme. Yoga will be expressed on the page with the use of the lotus flower, the yin yang sign and other yoga icons. We will be using SVG and creating simple CSS generated graphics for links. Some icons will display animation upon hover. The layout of the page will be in a simple block style with clean lines. This will produce a clean and easy to follow to the page. This page will be responsive to the screen size using a small, medium and large layout, as shown in the wireframes. Description of theme here. This layout is based off of the Mid Century Modern version of the Zen Garden project.
